Output Format

1 ≤ X,Y,Z ≤ 500

X!=Y, Y!=Z, X!=Z

Time Limit

1 second

Example

Sample Input

2 9 23

Sample Output

9

Code

        const findSndSmallest = (x,y,z) => {
            if(x>y && x < z || x>z && x< y){
              return x;
            }
            else if(y>x && y< z || y>z && y< x){
              return y;
            }
            else if(z>x && z< y || z>y && z< x){
              return z;
            }
        };

2 9 23
